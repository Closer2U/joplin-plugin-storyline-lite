# Note Title

```
[Artifact] Starfall Blade
```

# Note Body

```
---
storyline_codex: artifact
aliases: Starfall, The Blade
power: Cuts through any enchantment
---

A longsword forged from meteorite iron. It was given to Aldric Vane on the day
he became Warden of Thornfield. The blade has a faint blue sheen and hums
when near magical artifacts.

**History:** One of four Starfall weapons created during the First Age. The
others have been lost.

**In story:** Aldric carries it at all times. It is the only weapon known to
be able to separate fused crown shards.
```
